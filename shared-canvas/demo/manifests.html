WWA Demo 1

This is a first demo of the new SGA Shared Canvas Viewer for WWA

It only includes some documents from 'duk'; in particular I avoided documents with paste-ons while we work on the grid system.

What to look for

  • Hopefully this can be helpful for further proofing of the encoding
  • Likewise, please let me know if something you'd expect to be rendered in a certain way is rendered incorrectly or not at all
  • Try out the different "views":
    •    shows only the image
    •    default side-by-side view
    •    shows only the text - this can be very helpful for texts with columns

Known issues, in progress

  • FIXED Marginalia on the left and right should be positioned at the right place, but will not yet scroll with a scrollable section
  • FIXED Marginalia sections may be scrollable - this should not happen and scroll bars will not appear eventually
  • Font sizes may be too big in some boxes
  • Metadata is not loaded from the header yet
  • Resizing the window might cause positioning problems
  • Lines are not rendered (working towards vertical lines on the side, but not sure about curved lines)
  • Images are missing except for duk.00055, duk.00200 and duk.00679, but they're not zoomable (the zoom code is ready, but we need a Djatoka server and JP2 images)
  • Image proportions are not exact (I did not use the image list)
